# AutoClicker

지정한 화면 좌표를 정해진 시간 후 자동 클릭해 주는 콘솔 프로그램입니다.
초보자도 README만 보고 실행할 수 있도록 만들었습니다.

대표 사용 예: **"2시간 뒤에 (500, 700) 좌표를 6번 자동 클릭"**

---

## 무엇을 할 수 있나요?

### 콘솔 모드 (`main.py`)
- 메뉴 기반: 좌표 확인 / 예약 클릭 / 종료
- 1분마다 남은 시간 표시, 마지막 10초 1초 단위 카운트다운
- 입력 검증 (문자/음수 재입력)

### GUI 모드 (`gui.py`, **권장**)
- **마우스 위치 라이브 미리보기** (200ms 갱신, 캡처 없이도 위치 확인)
- **좌표 입력** + "3초 후 마우스 좌표 캡처" 버튼
- **시작 시점**: 상대(지금부터 N시간 N분) / 절대(오늘 HH:MM, 지난 시각이면 내일 자동 예약)
- **클릭 종류**: 좌클릭 / 우클릭 / 더블클릭
- **자동 절전 방지** (Windows `SetThreadExecutionState` — 대기 중 절전/화면 꺼짐 차단)
- **사운드 알림** (`winsound` — 클릭 시작 직전 / 완료 시 beep)
- **진행 Progress Bar** + 상태 라벨 + 로그 영역
- **프로파일 JSON 저장/로드** (시작 시 자동 적용)
- **중단 버튼** + fail-safe (마우스를 화면 모서리로) + 안전한 창 닫기

## 요구사항

- Windows 10/11 권장 (macOS, Linux도 동작 가능)
- Python 3.10 이상

## 설치

### 1) Python 설치 확인
PowerShell 또는 CMD를 열고:
```
python --version
```
`Python 3.10.x` 이상이 출력되지 않으면 [python.org](https://www.python.org/downloads/) 에서 설치하세요.
설치 시 **"Add Python to PATH"** 옵션을 꼭 체크하세요.

### 2) 의존성 설치

**가장 쉬운 방법**: 프로젝트 폴더의 `install.bat` 을 **더블클릭**하면 자동 설치됩니다.

또는 PowerShell / CMD 에서 직접:
```
py -m pip install -r requirements.txt
```
(`py` 가 없으면 `python -m pip install -r requirements.txt`)

## 실행

Windows에서는 **`.bat` 파일을 더블클릭**하는 것이 가장 쉽습니다:
- `run_gui.bat` → GUI 모드
- `run_console.bat` → 콘솔 모드

또는 PowerShell / CMD에서 직접 실행:

### 콘솔 모드
```

```

```
python gui.py
```

## GUI 사용법 (gui.py)

### 화면 구성
| 영역 | 설명 |
|------|------|
| 현재 마우스 위치 | 200ms 마다 좌표 라이브 갱신 (캡처 안 해도 위치 확인 가능) |
| 클릭 좌표 | x/y 직접 입력 또는 "3초 후 캡처" 버튼 |
| 시작 시점 | 상대(N시간 N분 뒤) ↔ 절대(오늘 HH:MM) 라디오 토글 |
| 클릭 설정 | 좌/우/더블 라디오 + 횟수 + 간격(초) |
| 옵션 | 자동 절전 방지 (Windows) / 사운드 알림 체크박스 |
| 액션 | 예약 시작 / 중단 / 설정 저장 / 설정 불러오기 |
| 진행 | 상태 라벨 + Progress Bar |
| 로그 | 모든 단계 기록 (스크롤) |

### 사용 흐름
1. **좌표 잡기**: 라이브 미리보기 라벨로 위치 확인 또는 "3초 후 캡처" 버튼.
2. **시작 시점**: 상대(예: 2시간 0분) 또는 절대(예: 오늘 17:30) 선택.
3. **클릭 종류 / 횟수 / 간격** 설정.
4. (선택) **자동 절전 방지** ON — 2시간 대기 중 Windows 절전/화면 꺼짐 차단.
5. (선택) **사운드 알림** ON — 클릭 직전/완료 시 beep.
6. **예약 시작** → 확인창의 요약을 보고 "예".
7. 카운트다운 (1분 단위 → 마지막 10초 1초 단위, Progress Bar 동기 갱신).
8. 자동으로 마우스 이동(1초) → 지정한 클릭 종류로 N회 실행.
9. 자주 쓰는 설정은 **설정 저장**으로 `autoclicker_profile.json` 에 보관 → 다음 실행 시 자동 적용.

### 단축키 (창 포커스 시)
- **Ctrl + Enter**: 예약 시작
- **Esc**: 중단
- **F8**: 3초 후 마우스 좌표 캡처

### 중단 방법
- **중단 버튼** 또는 **Esc**: 카운트다운 / 클릭 진행 중 즉시 중단
- **마우스 → 화면 모서리(좌상단)**: PyAutoGUI fail-safe 트리거
- **창 닫기 (X)**: 진행 중이면 확인창 후 종료 (절전 차단도 자동 해제)

## 콘솔 사용법 (main.py)

### 메뉴 1: 현재 마우스 좌표 확인
1. `1` 입력 후 Enter
2. 클릭하고 싶은 위치로 마우스를 옮김
3. 3초 카운트다운(3, 2, 1) 후 좌표가 출력됨
4. `x`, `y` 값을 메모해 두기

### 메뉴 2: 예약 클릭 실행
1. `2` 입력 후 Enter
2. 다음 값을 차례로 입력 (Enter만 누르면 기본값 적용):
   - **x 좌표** : 메뉴 1에서 확인한 x
   - **y 좌표** : 메뉴 1에서 확인한 y
   - **대기 시간 (시간)** : 기본 2
   - **대기 시간 (분)** : 기본 0
   - **클릭 횟수** : 기본 6
   - **클릭 간 간격(초)** : 기본 0.2
3. 입력 요약을 확인한 뒤 `y` 입력
4. 1분마다 남은 시간이 표시되고, 마지막 10초는 1초 단위로 카운트
5. 자동으로 마우스가 좌표로 이동(1초 동안) 후 6회 클릭

### 메뉴 3: 종료
- `3` 입력

## 좌표 시스템

- **(0, 0)** 은 화면의 **왼쪽 위 모서리**입니다
- **x** 는 오른쪽으로 갈수록 커집니다
- **y** 는 아래로 갈수록 커집니다
- 메뉴 1로 직접 좌표를 확인하는 게 가장 정확합니다

## 주의사항

### 절전 모드 / 화면 잠금
긴 대기 시간 동안 Windows 절전 모드나 화면 잠금이 발동되면 클릭이 제대로 동작하지 않을 수 있습니다.
- **Windows 설정 → 시스템 → 전원 → 화면 끄기 / 절전 모드** 를 "안 함"으로 설정 권장
- 또는 [Caffeine](https://www.zhornsoftware.co.uk/caffeine/) 같은 절전 방지 유틸리티 사용

### Fail-safe (안전 중단)
PyAutoGUI는 마우스가 화면 모서리(특히 좌상단)에 닿으면 즉시 동작을 중단합니다.
실수로 자동 동작이 시작됐거나 멈추고 싶을 때 **마우스를 좌상단으로 빠르게 이동**하면 멈춥니다.

### 관리자 권한이 필요한 창
관리자 권한으로 실행되는 프로그램의 창은 일반 권한의 PyAutoGUI가 클릭할 수 없습니다.
이 경우 **PowerShell 또는 CMD를 "관리자 권한으로 실행"** 한 뒤 `python main.py` 를 실행하세요.

### DPI 스케일링
모니터 배율이 100%가 아닐 경우 좌표가 어긋날 수 있습니다.
**메뉴 1로 좌표를 직접 확인**한 후 사용하세요.

## FAQ

**Q. 좌표가 음수로 나옵니다.**
A. 보조 모니터를 주 모니터의 왼쪽이나 위에 배치한 경우 발생합니다. 정상이며, 그대로 사용해도 됩니다.

**Q. 도중에 멈추고 싶어요.**
A. 마우스를 화면 좌상단으로 빠르게 이동시키거나, 콘솔 창에서 `Ctrl+C` 를 누르세요.

**Q. 더블 클릭/우클릭도 되나요?**
A. 현재 버전은 단일 좌클릭만 지원합니다.

**Q. 클릭이 6번 되긴 하는데 너무 빨라요/느려요.**
A. "클릭 간 간격(초)" 값을 조절하세요. `0.5` 면 0.5초마다, `1.0` 이면 1초마다 클릭합니다.

**Q. 한 번에 여러 다른 좌표를 클릭할 수 있나요?**
A. 현재 버전은 동일 좌표 N회 클릭만 지원합니다. 여러 좌표가 필요하면 매번 메뉴 2를 다시 실행하세요.

## EXE 단일 파일 빌드 (배포용, 선택)

Python 미설치 PC에 배포하고 싶다면 [PyInstaller](https://pyinstaller.org/)로 단일 EXE 생성 가능.

### 가장 쉬운 방법: `build.bat` 더블클릭
- pyinstaller 자동 설치 (없을 시)
- 이전 build/dist 정리
- `dist\AutoClicker.exe` 생성

### 또는 수동 빌드
```
pip install -r requirements-dev.txt
pyinstaller --noconfirm --onefile --windowed --name AutoClicker gui.py
```

빌드 결과: `dist\AutoClicker.exe` (단일 파일, 더블클릭 실행 가능).

> 일부 백신이 PyInstaller 단일 EXE를 의심할 수 있습니다 (잘 알려진 false positive). 배포 시에는 코드 서명을 적용하거나 신뢰된 PC에서만 사용하세요.
